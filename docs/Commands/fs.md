hg fs

control the edenfs daemon

Start and stop the edenfs daemon:

start start the edenfs daemon stop stop the edenfs daemon restart
restart the edenfs daemon

Maintenance for the edenfs daemon:

check check the filesystem of a virtual checkout doctor debug and fix
issues with the edenfs daemon gc minimize disk and memory usage by
freeing caches

Other Subcommands:

chown change the ownership of a virtual checkout config show the edenfs
daemon configuration info show details about the virtual checkout list
list available virtual checkouts prefetch prefetch content for files
remove remove a virtual checkout stats print statistics for the edenfs
daemon status check the health of the edenfs daemon top monitor virtual
checkout accesses by process version show version information for the
edenfs daemon
